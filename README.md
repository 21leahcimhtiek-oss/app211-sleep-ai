# SleepCoach AI

> AI-powered sleep optimization

## Features
- Sleep analysis
- Bedtime routines
- Dream journal
- Wake optimization
- Sleep score

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- OpenAI GPT-4o-mini
- Stripe Subscriptions
- Vercel deployment

## Quick Start

```bash
npm install
cp .env.example .env.local
# Fill in API keys
npm run dev
```
